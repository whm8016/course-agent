# 电路基本概念

## 电路的组成

电路由以下基本部分组成：
- **电源**：提供电能（如电池、发电机）
- **负载**：消耗电能（如电阻、电灯、电动机）
- **导线**：连接各元件，传输电能
- **开关**：控制电路通断

## 基本物理量

### 电流（Current, I）
- 定义：单位时间内通过导体截面的电荷量
- 公式：$I = \frac{dq}{dt}$
- 单位：安培（A）
- 方向：规定正电荷移动方向为电流方向

### 电压（Voltage, U）
- 定义：电场力将单位正电荷从一点移到另一点所做的功
- 公式：$U = \frac{dW}{dq}$
- 单位：伏特（V）
- 电压是产生电流的原因

### 电功率（Power, P）
- 定义：单位时间内电路消耗或产生的电能
- 公式：$P = UI$
- 单位：瓦特（W）

## 欧姆定律

### 部分电路欧姆定律
$$U = IR$$

其中 R 为电阻，单位为欧姆（Ω）。

### 全电路欧姆定律
$$I = \frac{\varepsilon}{R + r}$$

其中 ε 为电动势，r 为内阻。

## 电阻的串并联

### 串联
$$R_{总} = R_1 + R_2 + \cdots + R_n$$

特点：电流处处相等，电压按电阻分配。

### 并联
$$\frac{1}{R_{总}} = \frac{1}{R_1} + \frac{1}{R_2} + \cdots + \frac{1}{R_n}$$

特点：各支路电压相等，电流按电导分配。

两个电阻并联的简化公式：
$$R_{总} = \frac{R_1 \cdot R_2}{R_1 + R_2}$$
