# 基尔霍夫定律

## 基尔霍夫电流定律（KCL）

### 内容
对于电路中的任意节点，流入节点的电流之和等于流出节点的电流之和。

$$\sum I_{入} = \sum I_{出}$$

或者等价地写成：

$$\sum_{k=1}^{n} I_k = 0$$

（规定流入为正，流出为负）

### 物理本质
KCL 本质上是电荷守恒定律的体现——节点处不会积累电荷。

### 应用示例
一个节点有三条支路，电流分别为 $I_1$、$I_2$、$I_3$。若 $I_1 = 2A$（流入），$I_2 = 3A$（流入），则：
$$I_3 = I_1 + I_2 = 5A（流出）$$

## 基尔霍夫电压定律（KVL）

### 内容
对于电路中的任意闭合回路，沿回路方向所有电压降的代数和为零。

$$\sum_{k=1}^{n} U_k = 0$$

### 物理本质
KVL 本质上是能量守恒定律的体现——电荷沿闭合回路运动一周，电位回到原值。

### 应用步骤
1. 选定回路绕行方向（顺时针或逆时针）
2. 对回路中每个元件列写电压
3. 与绕行方向一致的电压降取正号
4. 与绕行方向相反的电压降取负号
5. 所有电压代数和为零

### 应用示例
一个简单回路包含电源 E 和两个电阻 $R_1$、$R_2$，设电流为 I：
$$E - IR_1 - IR_2 = 0$$
$$I = \frac{E}{R_1 + R_2}$$

## 用 KCL 和 KVL 解题的一般方法

1. 标注支路电流的参考方向
2. 对独立节点列 KCL 方程（节点数 - 1 个）
3. 对独立回路列 KVL 方程
4. 联立求解方程组
5. 若某电流求得为负值，说明实际方向与参考方向相反
