# 中国邮票发展史

## 清代邮票（1878-1911）

中国第一套邮票是1878年发行的**大龙邮票**，全套三枚，面值分别为一分银、三分银和五分银。图案为蟠龙戏珠，由上海海关造册处印制。

其后陆续发行：
- **小龙邮票**（1885年）：图案与大龙类似但尺寸较小
- **万寿邮票**（1894年）：慈禧太后六十寿辰纪念
- **蟠龙邮票**（1897年起）：清代使用时间最长的邮票

## 民国邮票（1912-1949）

民国时期邮票种类丰富：
- 孙中山像普通邮票：使用最广泛
- 各类纪念邮票：纪念国家大事
- 航空邮票：随航空事业发展而出现

## 新中国邮票（1949至今）

### 编号体系演变
1. **纪特邮票**（1949-1967）：用"纪"和"特"字头编号
2. **文革邮票**（1967-1970）：取消志号，称"文"字邮票，存世量少，收藏价值极高
3. **编号邮票**（1970-1974）：使用连续编号
4. **J/T邮票**（1974-1992）：J表示纪念，T表示特种
5. **编年邮票**（1992至今）：以年份编号，沿用至今

### 经典系列
- **生肖邮票**：1980年起每年发行，首枚"庚申年猴票"（T46）是集邮界传奇，面值8分，现价值数万元
- **名山大川系列**：展示中国壮丽河山
- **古典文学名著系列**：红楼梦、三国演义、水浒传、西游记等

## 邮票收藏价值评估

影响邮票价值的因素：
1. **发行量**：数量越少越珍贵
2. **存世量**：实际留存数量
3. **品相**：全新未使用（新票）价值远高于盖销票
4. **题材**：重大历史事件或文化主题更受青睐
5. **错票**：印刷错误的邮票反而可能极具价值
